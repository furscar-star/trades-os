# Trades-OS — Vision

## 한 줄

소규모 노동력 중심 사업자가 **현장에서도 한 번의 질문으로** 작업·일정·매출을 파악하게 한다.

## 타깃 페르소나

- 1~10인 소상공 시공/수리 사업자
  - **핸디맨** (handyman, 잡탕 수리)
  - **종합 시공사** (general contractor, 리노베이션)
  - **전기공** (electrician, ESA 라이센스)
  - **배관공** (plumber)
  - **HVAC** 기술자
  - **청소·조경·세차** 등 노동력 중심
- GTA(토론토 권역) 한인·중국계·영어권 동시 타겟

## 페인 포인트

1. **들어오는 문의를 다 못 받는다** — 현장 작업 중엔 전화 못 받음 → 리드 손실
2. **누가 어디서 뭐 하는지 머리속에만 있음** — 작업 진행 상황 흩어짐
3. **이번 달 얼마 벌었지** 가 즉답 안 됨 — 종이/엑셀 흩어짐
4. **다음 사이트 가는 데 얼마 걸리지** 일일이 구글맵 — 점심·이동시간 못 챙김

## 우리가 푸는 방식

| 페인 | 솔루션 | 핵심 기술 |
|---|---|---|
| 부재중 통화 | 인바운드 음성봇 (다국어) | Vapi + Whisper |
| 작업 추적 | 잡 관리 대시보드 + AI 비서 자연어 | Supabase + Groq |
| 매출 즉답 | DB 집계 → 비서 응답 | 비서 툴 |
| 동선 최적화 | 자연어로 이동시간/순서 질의 | ORS + 비서 |

## 차별점 (vs Jobber, ServiceTitan)

- **음성 인터페이스 1차** — 손에 페인트·기름이 묻어 있어도 운영 가능
- **다국어 1차** — 영어 외 한국어/중국어 모국어 사장님 대상
- **MVP 베타 무료** — Jobber 월 $69, ServiceTitan은 엔터프라이즈

## 비전

Phase 5 즈음:
- 음성 → 자동 견적 초안 → 고객 SMS/메일 발송
- 작업 후 사진 → AI 진행률 산정 → 인보이스 자동
- 직원 출퇴근 자동 (위치/QR)
- 다중 사이트 동시 운영자에게 "지금 어디 가야 하지" 추천
