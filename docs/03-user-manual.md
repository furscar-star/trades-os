## 3. 잡(공사) 관리

### 상태 흐름
```
lead(문의) → quoted(견적) → scheduled(예약) → in_progress(진행 중) → completed(완료)
                                ↓
                            cancelled(취소)
```

### 잡 카드 클릭하면 모달 열림 — 다음 항목 편집 가능
- 작업명·고객명·연락처·주소
- 유형 (핸디맨/시공/전기/배관/HVAC/청소/조경/기타)
- 상태 (위 6개 중)
- **예상 매출 / 예상 비용 (CAD)**
- **투입 인원 / 배정 인력** (예: "Mike, Joon")
- **시작일 / 종료 예정**
- 상세 설명

### 신규 잡 추가
신규 의뢰 카드 우측 상단 **+ 추가** → 빈 모달 → 저장.

### 자동 저장
모든 변경은 즉시 Supabase에 저장. 30초마다 자동 새로고침.

## 4. 이번 달 매출 카드

| 항목 | 의미 |
|---|---|
| **파이프라인** | 이번 달 등록된 모든 잡의 예상매출 합계 (lead·quoted 포함) |
| **진행중·완료** | 진행중 + 완료 잡의 매출 합계 (실현된 매출에 가까움) |
| **예상 비용** | 이번 달 등록된 모든 잡의 예상 비용 합계 |
| **이번 달 잡** | 등록된 잡 건수 |

\*파이프라인 - 진행중·완료 = 아직 확정 안 된 잠재 매출.

## 5. AI 비서 (💬 우측 하단)

자연어로 잡 관리·매출 조회·이동시간·일정 등록까지.

### 조회 예시
- "오늘 진행 중인 작업"
- "Smith 댁 일정"
- "이번 달 매출 합계"
- "전기 작업만 보여줘"
- "Hot 고객 누구야" *(잡 상태가 lead/quoted/scheduled인 것)*

### 기록·등록 예시 (확인 단계 거침)
- "Smith 댁 작업 완료로 변경"
  → "이렇게 변경할게요: Smith 욕실 → completed. 진행할까요?" → 사용자 "네" → 실제 변경
- "내일 오전 9시 Johnson 거실 페인트 등록"
  → 비서가 ISO 형식으로 시각 환산 → 확인 요청 → 등록
- "이지훈 잡에 Mike랑 Joon 배정"
  → 확인 → 잡의 assigned_crew 갱신

### 이동시간
- "리치먼드힐에서 마캄 ABC까지 얼마"
- "오늘 가야할 3곳 순서 짜줘" → optimize_route 툴이 최단 경로 계산

\*ORS 무료티어 사용. 실시간 교통 미반영, 러시아워 +20% 안전 버퍼 권장.

### 음성 입력 🎤
챗 입력란 좌측 마이크 버튼 → 클릭하면 녹음 → 다시 클릭하면 정지·텍스트 변환·입력란 자동 채움. (운전 중·페인트 묻은 손으로도 사용 가능)

### 핀(고정) 📌
챗 패널 헤더 📌 클릭 → 노란색 📍 → 페이지 새로고침해도 패널 유지. 다시 클릭하면 해제.

## 6. 인력 관리

DB의 `crew` 테이블에 미리 4명 등록 (Mike Chen / Joon Park / Kim Lee / Tony Wang).

대시보드 UI엔 아직 인력 관리 화면 없음 — **챗으로 조회**:
- "인력 명부 알려줘" → list_crew 툴

V2에서 별도 화면 추가 예정.

## 7. 메모

대시보드 우측 하단의 메모 카드. 텍스트 입력 후 + 추가. 챗으로도 가능 — "메모 추가: ABC 자재 5/15 도착 확인".

## 8. 다국어

우측 상단 드롭다운: 한국어 / English / 中文. 즉시 전환, localStorage 저장 (다음 방문 시 같은 언어).

**번역이 아니라 현지 표현**:
- KR: "신규 의뢰" / "진행 중" / "이번 달 매출"
- EN: "New Requests" / "In Progress" / "Monthly Revenue"
- ZH: "新订单" / "施工中" / "本月收入"

\*고객 데이터 자체(예: "Smith 욕실 타일 교체")는 등록된 그대로 표시. 자동 번역 안 함.

## 9. 데이터

샘플 데이터 4개 잡 + 4명 인력 + 3개 메모 사전 입력됨:
- Smith 욕실 타일 교체 (시공/in_progress, $4,800)
- Wong 콘도 거실 페인트 (시공/scheduled, $2,200)
- Lee 오피스 LED 교체 (전기/lead, $3,500)
- Park 단독주택 에어컨 점검 (HVAC/quoted, $650)

데이터 영속: Supabase Postgres에 저장. 새로고침해도 유지.

## 10. 미설정 항목 (다음 단계)

| 항목 | 상태 | 어떻게 |
|---|---|---|
| Vapi 음성봇 번호 | ❌ 신규 발급 필요 | Vapi Dashboard → 새 Phone Number → 새 Assistant 생성 → 시스템 프롬프트 다국어 설정 |
| Google Calendar/Gmail | ❌ env 미설정 | GCP Console에서 새 OAuth Web Client 만들고 Redirect URI 추가 (`https://trades-os-nine.vercel.app/oauth/google/callback`) |
| Telegram 봇 | ❌ 토큰 미발급 | @BotFather → /newbot → 토큰 → Vercel env `TELEGRAM_BOT_TOKEN` |
| Twilio SMS | ❌ env 미설정 | Twilio 가입 → Vercel env 설정 |

핵심 1·2·3·5·8 기능은 위 4개 없이도 작동. 1차 검토 충분.
